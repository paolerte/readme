Acerca de esta cuenta

  • Este repositorio contiene las instrucciones necesarias para replicar el curso Taller de R: Estadstica y Programacin..

1. Acerca de este curso

Qu es? y Qu no es?

  • Este no es un curso dedicado a hacer econometra usando R. A lo largo del curso empleamos distintos conceptos estadsticos, pero la econometra no es el core del curso.

  • Este no es un curso avanzado de R. Desde las primeras las clases, el curso est pensado para personas que nunca han tenido contacto con la programacion.

  • Este curso est diseado para introducir a los usuarios en mtodos computacionales para la investigacin econmica.

Algunos comentarios adicionales

  • Aunque el curso est desarrollado completamente en espaol, no se usan acentos ni caracteres especiales dentro de los scripts de las clases; Esto ltimo para evitar conflictos entre los diferentes sistemas operativos.

  • Los comentarios para mejorar el contenido del curso son bienvenidos.

2. Acerca de las clases

  • Para replicar las clases del curso, usted deberia revisar los videos 1 y 2 de este enlace y en este otro enlace encuentra 3 formas de clonar un repositorio de GitHub usando R. Los vdeos de clases se encuentran en este [link]

Para cada clase se crea un repositorio en la cuenta del curso en GitHub. Cada repositorio contiene 6 carpetas y un archivo README.md que contiene las instrucciones iniciales de la clase. En este archivo tambin se encuentran los link a los videos de la clase. Los vdeos de las clases (incluidos los de este repositorio) se encuentran organizados por carpetas aqu.

Ademas, el repositorio contiene 6 carpetas con el material de la clase:

  • help:

Esta carpeta contiene 1 carpeta con los grficos de apoyo para la clase y 2 archivos (.Rdm y .html).

El archivo .Rdm contiene el cdigo de Rmarkdown para generar el archivo .html de la clase (este ultimo es el que realmente importa).

El archivo .html contiene el material de apoyo para la clase. Debe abrir este archivo (haciendo doble clic sobre el) en paralelo con el script de la clase. En este archivo, generalmente se muestran grficos que ayudan a entender la intuicin de lo que se est haciendo el script de la clase. En este archivo tambin se encuentran los link a los videos de la clase.

  • codes:

Esta carpeta contiene el script (editor de sintaxis) que vamos a usar en la clase.

  • data:

Esta carpeta contiene dos carpetas (originals y processed) con las bases de datos que vamos a usar en la clase. La carpeta originals contiene la raw data (datos originales), es decir la informacin tal cual como se obtiene de la fuente original. La cual ser procesada con el script de la clase. Por su parte, la carpeta processed contiene la tidy data (datos procesados), es decir, las bases de datos que se obtiene al correr el script de la clase.

  • Results:

Esta carpeta generalmente contiene algunos de los grficos que se producen en el script de la clase.

  • Task:

Esta carpeta contiene un archivo .html con algunas instrucciones para realizar ejercicios de los temas vistos en clase. La intencin es que los estudiantes puedan afianzar los conocimientos de la clase empleando bases de datos de instituciones como DANE.

Posteriormente se incluir otro archivo .html adicional con las respuestas a los ejercicios.
